Element

Element är en komponent innom HTML kod som bland annat strukterar upp koden, börjar oftast med en starttag och slutar med en end tag.

HTML

HTML eller HyperText Markup Language är egentligen ett "kod" språk och majoriteten om inte alla webbsidor består av html kod och dess struktur.

ID

ID är en sorts namngivning för kod, man ger en specifik kod ett alias och kan därefter ändra endast den koden.
En html kod kan bara he en identisk ID

Class

Class liknar ID men det kan finnas flera aliaser med samma klass så att du kan ändra all kod med samma alias samtidigt med mindre kod.

Meta

Meta används i webbsidor för att bestäma hur saker skall strukteraras, skall alltid ligga i head.

Responsive

Responsive används exempel för att göra webbsidor mobil anpassande genom att automatiskt ändra elemets storlek.

Http

HyperText transfer protocol är en slags kommunikation mellan olika webbsidor.
Används även för att få webbsidor från servrar till användare.

Https

Https eller HyperText transfer protocol secure är nästan samma sak som http men mer säker därav "secure".
Gör samma sak men mer eftersom den krypterar datan och ser till att den är säker.

Compressed data

Compressed data gör exakt vad namnet säger, den kompressionerar data altså får data att ta mindre plats.
Compressed data skriver om koden på ett smartare sett så mer kod tar mindre plats.

CDN

Content Dilivery Network existerar så att du kan använda internet utan extra väntetid.
CDN är ett utspritt nätverk som hjälper datan att komma från en server till en klient på kort tid.

Local bootstrap

Local bootstrap är en mall av css altså allt skaparna kunde tänka på i en enda css fil.
Kallas även för "biblotek" eftersom det har så mycket innehåll cirka 10 000 rader av kod.

JQuery

JQuery existerar för att underlääta HTML kod eftersom det är ett Javascript-bibliotek.

Flexbox

Flexbox är förinprogramerade kommandon som kan ändra utseendet på webbsidor exempel "justify-content:center;"
vilket som då centrerar objektet.

Grid

Grid är en slags webbsida som är indelad i olika delar, (se bild nedan) har en header, footer.
Kan även användas föt att placera objekt i kolumner eller i rader, placera objekt i olika mönster.

Code snippets

Code snippets är tricks som gör att du kan koda snabbare genom att "återanvända kod".
Code snippets kan även sparas som valfri kod och sedan återanvändas.
